# dbt 學習路徑

本文件依照學習步驟，帶領你從零開始掌握 dbt 並應用於半導體公司採購部門的資料專案。

1. 什麼是 dbt？（dbt 核心概念、優勢、應用場景）
2. 安裝與環境設定（安裝步驟、profiles.yml 範例、常見問題）
3. 專案初始化與結構說明（目錄說明、初始化範例）
4. 匯入範例資料（Seed）（seed 格式、應用、驗證）
5. 撰寫與執行 Models（命名慣例、依賴、Jinja、分層）
6. 撰寫與執行 Tests（自訂測試、schema.yml 範例、排查建議）
7. 撰寫 Macros 與進階應用（macro 範例、呼叫、應用）
8. 產生與瀏覽文件（description 撰寫、docs serve 技巧）
9. 常見問題與資源（錯誤排查、產業挑戰、進階學習）
